# 删除分支

> ` localBranchName` 是你要删除的分支的名字

##  本地

```
git branch -d localBranchName
```
## 远程
```
git push origin --delete remoteBranchName
```

# 推送本地分支到远程
> ` localBranchName` 是你要推送的本地分支的名字
```
git push --set-upstream origin localBranchName
```
